# 🗡️ Focus: Ritual of Debt (Pixel Art Edition)

## 🎮 Game Flow (The Ritual)
1. **Main Menu:** [Comenzar Ritual] [Dominios] [Bestiario] [Trofeos] [Iniciación].
2. **Setup Ritual:** - Select Offering (Time: 1h to 8h).
   - Select Domain (Subject to apply time).
   - Select Beast (The boss to fight).
3. **The Battle:** Countdown timer. Visual boss HP = Remaining time.
4. **The Bonefire (Rest):** 5-min rest gif after session. Option to +5 min or Skip.
5. **Victory:** Update Bestiary, add XP, reduce Weekly Debt of the Domain.

## 🏛️ Domain System (Subjects)
- Max 50 Domains. 
- Stats: Total minutes (Display as "XXh YYm"), Mastery Bar (Progress to 1000h).
- Weekly Debt: If current week minutes < weekly target, debt accumulates.

## 👾 Bestiary & Trophies
- Track kills per beast. 
- Trophy logic: "First 100h", "Kill Alberic 3 times", etc.

## 🎨 Aesthetics
- Pixel Art (rounded-none, border-4, shadow-pixel).
- Font: 'Press Start 2P'.
- Animations: Step-based. Sounds: start-battle, bonefire, bell.
